## About this file

Suggest Edits
It contains 27,481 tweets extracted using the twitter api. The tweets have been annotated (negative, neutral, positive) and they can be used to detect sentiment .
It contains the following 4 fields:

textID: The id of the tweet
text: the text of the tweet (Lyx is cool)
selected_text: text selected on the basis of sentiment
sentiment: neutral, negative & positive
